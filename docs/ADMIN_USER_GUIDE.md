## 1. บทนำ: ทำความรู้จักหน้าจอหลัก

เมื่อคุณ Login เข้าระบบมาแล้ว จะเจอ **"Dashboard"** เปรียบเสมือนห้องทำงานส่วนตัว

### �️ ส่วนประกอบหน้าจอ
1. **แถบเมนูซ้ายมือ (Sidebar)**: เครื่องมือทั้งหมดอยู่ที่นี่ แบ่งเป็น 5 หมวด (ที่เราจะเรียนกัน)
2. **กล่องสรุปข้อมูล (Widgets)**: ตัวเลขใหญ่ๆ ด้านบน
   - *New Users*: วันนี้มีคนทักมาใหม่กี่คน?
   - *Pending Orders*: มีกี่ออเดอร์ที่รอเราไปจัดการ? (ถ้าเลขไม่ใช่ 0 ต้องรีบกดดู!)
   - *Sales*: วันนี้ขายได้กี่บาทแล้ว?
3. **ปุ่มทางลัด (Quick Access)**: ปุ่มสีๆ ตรงกลาง คือเมนูที่ใช้บ่อย กดเข้าได้เลยไม่ต้องหา

> **💡 ทริค**: ถ้าคุณเป็นแอดมินตอบแชท ให้เปิดหน้า **"Inbox"** (กล่องข้อความ) ค้างไว้ตลอดเวลา

## 2. หมวดที่ 1: การดูแลลูกค้าและตอบแชท (Patient & CRM)
*(เมนู: ดูแลลูกค้า)*

งานหลักของแอดมินคือ "ตอบแชท" และ "ดูแลลูกค้าเก่า"

### � 2.1 สอนตอบแชท (Inbox)
เหมือนเล่น LINE หรือ Facebook แต่เก่งกว่า!

**วิธีใช้งาน:**
1. กดเมนู **"กล่องข้อความ"**
2. **ด้านซ้าย**: คือรายชื่อลูกค้า
   - จุดสีเขียว 🟢 = ลูกค้ากำลังออนไลน์ (รีบตอบเลย!)
   - ตัวเลขสีแดง 🔴 = ข้อความใหม่ที่ยังไม่ได้อ่าน
3. **ตรงกลาง**: พื้นที่แชท พิมพ์คุยได้เลย
4. **ด้านขวา**: "โพย" ข้อมูลลูกค้า
   - จะเห็นเลยว่าลูกค้าคนนี้ชื่ออะไร, เป็นสมาชิกระดับไหน (Gold/Silver), เคยซื้ออะไรไปบ้าง

**ฟีเจอร์เด็ดที่ต้องใช้:**
- **ส่งสินค้า**: อย่าส่งแค่รูป! ให้กดไอคอน 🛍️ แล้วเลือกสินค้า ลูกค้าจะเห็นปุ่ม "กดใส่ตะกร้า" ได้เลย (ซื้อง่ายขึ้นมาก)
- **Quick Reply**: คำตอบที่พิมพ์บ่อยๆ เช่น เลขบัญชี, รอบส่งของ ให้ save เก็บไว้ กดปุ่ม ⚡ เลือกส่งได้เลยใน 1 วิ
- **Tag**: ลูกค้าคนนี้เป็นใคร? ติดป้ายบอกไว้ เช่น `VIP`, `เรื่องมาก`, `แพ้ยา` ครั้งหน้ามาคุยจะได้ระวังตัวถูก

### 📢 2.2 สอนบรอดแคสต์ (Broadcast)
คือการส่งข้อความหาลูกค้า "ทุกคน" พร้อมกัน (เหมือนประกาศเสียงตามสาย)

**วิธีใช้งาน:**
1. กดเมนู **"บรอดแคสต์"**
2. กด **"+ สร้างบรอดแคสต์ใหม่"**
3. **ขั้นที่ 1 เลือกคนรับ**:
   - ถ้ามีโปรโมชั่นทั่วไป -> เลือก "ทุกคน"
   - ถ้าโปรฯ นมผง -> เลือกเฉพาะ Tag "แม่ลูกอ่อน" (อย่าส่งมั่ว ลูกค้าจะรำคาญแล้ว Block เรา)
4. **ขั้นที่ 2 ใส่ข้อความ**: พิมพ์ข้อความ หรือใส่รูปภาพโปรโมชั่น
5. **ขั้นที่ 3 ส่ง**: จะกดส่งทันที หรือ "ตั้งเวลา" ให้ส่งพรุ่งนี้เช้าก็ได้

### 💳 2.3 สอนจัดการสมาชิก (Membership)
ลูกค้าทุกคนที่ Add LINE มาจะเป้นสมาชิกขั้นต้นทันที

**สิ่งที่คุณทำได้:**
- **ลูกค้าถามแต้ม**: กดค้นหาชื่อลูกค้า -> ดูช่อง "Points" ตอบลูกค้าได้เลย
- **แก้แต้มให้ลูกค้า**: ถ้าลูกค้าบ่นว่าแต้มหาย หรืออยากแถมแต้มปลอบใจ ให้กดปุ่ม "Manage Points" -> ใส่จำนวน -> ใส่เหตุผล -> กดบันทึก (ลูกค้าจะได้รับ LINE แจ้งเตือนแต้มเด้งทันที)

## 3. หมวดที่ 2: การจัดการออเดอร์และคลังสินค้า (Supply)
*(เมนู: คลังสินค้าและยอดขาย)*

### 🧾 3.1 สอนจัดการออเดอร์ (Order Lifecycle)
เมื่อลูกค้าสั่งของมา ออเดอร์จะไหลไปตามขั้นตอนดังนี้ หน้าที่ของคุณคือ **"ดัน"** ออเดอร์ให้จบ

**ขั้นตอนที่ 1: ตรวจสลิป (Waiting Proof)**
- ไปที่เมนู **"รายการสั่งซื้อ"** -> แท็บ **"รอตรวจสลิป"**
- กดที่ออเดอร์ -> ดูรูปสลิป เทียบกับยอดเงิน
- ✅ ถ้าตรง: กด **"อนุมัติ" (Approve)**
- ❌ ถ้าไม่ตรง/สลิปปลอม: กด **"ปฏิเสธ" (Reject)**

**ขั้นตอนที่ 2: จัดส่ง (Shipping)**
- เมื่อแพ็คของเสร็จ ให้ไปที่แท็บ **"ที่ต้องจัดส่ง" (Paid)**
- กดปุ่ม **🚚 จัดส่ง**
- ใส่เลขพัสดุ (Tracking No.) เช่น `KER123456`
- กดบันทึก -> ระบบจะส่งข้อความ LINE ไปบอกลูกค้าเองว่า "ส่งของแล้วนะ นี่เลขพัสดุ..." (ไม่ต้องพิมพ์บอกเอง)

### 🛒 3.2 สอนขายหน้าร้าน (POS)
ถ้ามีลูกค้าเดินเข้ามาที่ร้าน (Walk-in) ให้ใช้เมนูนี้
1. กดเมนู **"POS ขายหน้าร้าน"**
2. **ยิงบาร์โค้ด** สินค้า หรือพิมพ์ค้นหา
3. ถามเบอร์ลูกค้า (ถ้ามี) เพื่อสะสมแต้ม
4. กด **"รับเงิน"** -> เลือกว่ารับ "เงินสด" หรือ "โอน"
5. เครื่องจะพิมพ์ใบเสร็จออกมา และตัดสต็อกทันที

### 📦 3.3 สอนลงสินค้าใหม่ (Inventory)
ของมาใหม่ ต้องเอาเข้าระบบก่อนถึงจะขายได้
1. กดเมนู **"คลังสินค้า"** -> **"สินค้า"** -> **"+ เพิ่มสินค้า"**
2. **ข้อมูลที่ห้ามลืมใส่**:
   - *ชื่อสินค้า*: เอาให้ลูกค้าอ่านเข้าใจง่าย
   - *SKU*: รหัสสินค้า (ถ้ามีบาร์โค้ด ให้ยิงใส่ช่องนี้เลย)
   - *ราคา*: ใส่ราคาขาย
   - *จำนวน*: มีของกี่ชิ้น?
   - *รูปภาพ*: สำคัญมาก! รูปสวยขายดี ใส่ได้สูงสุด 5 รูป
3. กดบันทึก

## 4. หมวดที่ 3: งานเภสัชกรรมและคลินิก (Clinical)
*(เมนู: งานบริการคลินิก)*
> **ส่วนนี้สำหรับเภสัชกร**

### 💊 4.1 สอนจ่ายยาและพิมพ์ฉลาก (Pharmacy Dispensing)
1. กดเมนู **"ห้องยา / จ่ายยา"**
2. เลือกชื่อคนไข้ (จาก LINE หรือ Walk-in)
3. พิมพ์เลือกยาที่จ่าย
4. **ระบบจะเตือนทันทีถ้า**:
   - ⚠️ คนไข้แพ้ยานี้
   - ⚠️ ยานี้ตีกันกับยาเดิมที่คนไข้กินอยู่ (Drug Interaction)
5. พิมพ์ฉลาก: กดปุ่ม Print -> สติ๊กเกอร์จะออกมาพร้อม "คำแนะนำการใช้ยา" อัตโนมัติ

### 📹 4.2 สอนทำ Telepharmacy (VDO Call)
ถ้าคุยแชทไม่รู้เรื่อง ต้องคุยเห็นหน้า
1. ไปที่เมนู **"นัดหมาย"** หรือหน้าแชทลูกค้า
2. กดปุ่ม **"Start Video Call"**
3. ระบบจะส่ง "Link" ให้ลูกค้าทาง LINE
4. เมื่อลูกค้ากด Link -> หน้าจอคุณจะเด้งขึ้นมาเห็นหน้าลูกค้า
5. **ระหว่างคุย**: พิมพ์บันทึกอาการ (Note) ด้านข้างได้เลย ข้อมูลจะเก็บเข้าประวัติคนไข้

## 5. หมวดที่ 4: การดูผลประกอบการ (Insights)
*(เมนู: ภาพรวมและสถิติ)*

เอาไว้ดูว่าร้านเราเจริญรุ่งเรืองแค่ไหน
- **Analytics**: ดูกราฟยอดขายรายเดือน, สินค้าตัวไหนขายดีสุด (Top Products)
- **Activity Logs**: เอาไว้จับโจร! ดูได้ว่าใครแอบแก้สต็อก, ใครลบออเดอร์, ใครทำอะไร เวลาไหน
