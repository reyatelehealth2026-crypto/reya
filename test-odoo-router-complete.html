🔄 Navigation Flow

Route Configuration

Routes defined in router.js with page names and titles

2

Handler Registration

Page handlers registered in liff-app.js registerPageHandlers()

3

Route Change Detection

Router detects navigation and calls onRouteChange()

4

Page Rendering

Render method returns initial HTML with loading state

5

Content Initialization

Init method fetches content from PHP and displays it

💻 Code Examples

JavaScript Navigation

// Navigate to orders list window.router.navigate('/odoo-orders'); // Navigate to specific order window.router.navigate('/odoo-order/123'); // Navigate to tracking window.router.navigate('/odoo-order-tracking/123'); // Go back window.router.back();

HTML Links

<!-- Link to orders --> <a href="#/odoo-orders">ดูออเดอร์ทั้งหมด</a> <!-- Link to specific order --> <a href="#/odoo-order/123">ดูรายละเอียด</a> <!-- Button navigation --> <button onclick="window.router.navigate('/odoo-orders')"> ออเดอร์ของฉัน </button>
